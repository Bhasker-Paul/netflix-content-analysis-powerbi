# 📊 Netflix Content Analysis Dashboard (Power BI)

This project presents an interactive Power BI dashboard analyzing Netflix content using a dataset sourced from Kaggle.

---

## 🔍 Project Overview

The goal of this project is to explore and visualize Netflix content to uncover key insights such as content distribution, growth trends, and audience targeting.

---

## 📈 Key Insights

- Total Titles: 8,700+
- Movies vs TV Shows distribution
- Content growth trend over the years
- Top 5 countries producing Netflix content
- Most common content ratings
- Top directors with highest number of titles
- Average duration by rating category

---

## 🛠 Tools & Technologies

- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning (Power Query)
- Data Visualization

---

## 📂 Dataset

The dataset used in this project is sourced from **Kaggle**:

Netflix Titles Dataset

---

## 📊 Dashboard Features

- Interactive slicers (Year, Country, Rating, Content Type)
- Top N analysis using DAX
- Multiple visualizations:
  - KPI Cards
  - Donut Chart
  - Line Chart
  - Bar Charts
